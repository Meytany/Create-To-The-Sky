# Create-To-The-Sky
Версия сборки CTTS для мощных ПК. Java 21. Оптимальное количество памяти: 12Гб (12288 мб).

## Инструкция по установке
# Prism Launcher
-Для установки в Prism Launcher скачайте архив CTTS Prism Launcher.rar
-Распакуйте архив в папку с установками.
-Перезапустите Prism Launcher.
# Обычный лаунчер(Legacy, TLauncher и подобные)
-Скачать CTTS Legacy.rar
-Распакуйте архив в папку, в которой у вас будет находиться майнкрафт.
-В настройках лаунчера укажите папку запуска, в которую распаковали архив.
-Выберите версию Minecraft 1.21.1 NeoForge 21.1.230 (или подобное название)
